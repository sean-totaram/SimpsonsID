# Simpsons Identifier

A simple image classifier using a convolutional neural network

### Prerequisites


* [Python 3.6](https://www.python.org/)
* [Tensorflow](https://www.tensorflow.org/)
* [Keras](https://keras.io/)
* [openCV](https://opencv.org/)
* [numpy](www.numpy.org/)
